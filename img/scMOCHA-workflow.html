Pipeline Architecture

scMOCHA

Single-Cell Mitochondrial Omics for Cellular Heteroplasmy Analysis — an end-to-end WDL pipeline for calling, annotating, and visualizing mitochondrial DNA variants from 10x Genomics single-cell RNA-seq data at both single-cell and cluster resolution.

3-in-1 Co-Assay
Simultaneous Multi-Modal Profiling: From a single scRNA-seq run, scMOCHA simultaneously calls Mitochondrial DNA Variants, quantifies Nuclear Gene Expression, and maps Cell Type Identities, linking heteroplasmy directly to phenotypic cell states.

▾
Process
Counts A, C, G, T nucleotides at every MT position (1–16,569) for every cell barcode. Tracks forward and reverse strand counts separately.

8
Cell Variant Processing
variant_calling_cell_raw.py
▾
Algorithm
  • AF = alt_reads / total_reads per cell per position
  • Min coverage ≥ 10 reads
  • Strand support on both strands (≥2 reads/strand)
  • Strand concordance: log₂(fwd/rev)
Outputs
Cell AF matrix Cell depth matrix Variant list

Runs mgatk on each cluster BAM file. Higher coverage per position (aggregated reads) enables more sensitive variant detection. Uses stricter strand filter (≥3 reads/strand).

Key Concepts

✨ Simultaneous calling (3-in-1 Co-Assay)

A primary scientific advantage of scMOCHA is its capability to **simultaneously profile host nuclear transcripts and mitochondrial genotypes from the same sequence library**. This allows researchers to call mitochondrial DNA variants and heteroplasmy, map cell-type identity, and analyze nuclear gene expression concurrently—saving experimental costs and preserving exact cellular context.

🧬 Heteroplasmy

The co-existence of multiple mtDNA genotypes within a single cell. scMOCHA quantifies this as allele frequency (AF) at each variant position per cell.

⑂ Dual-Resolution

Variants are called at both single-cell (cell-to-cell variation, lower coverage) and cluster level (aggregated reads, higher confidence).

📐 rCRS Standard

The Revised Cambridge Reference Sequence — the internationally agreed-upon reference for human mtDNA, ensuring consistent variant nomenclature (e.g., "73A>G").

✓ Strand Concordance

Quality metric checking if a variant is supported by reads from both forward and reverse strands, filtering out sequencing artifacts.

🔄 MT Genome

A circular, 16,569 bp genome encoding 13 protein-coding genes, 22 tRNAs, 2 rRNAs, and 1 non-coding D-loop control region.
